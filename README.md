# Tauri + React + Typescript

This template should help get you started developing with Tauri, React and Typescript in Vite.

## QA operativo

Antes de liberar cambios funcionales ejecuta:

```bash
npm run qa
```

La checklist manual de regresion esta en [docs/qa-regresion-operativa.md](docs/qa-regresion-operativa.md).
